Calculadora de Operações Básicas 

Calculadora completa com operadores matemáticos básicos.

Como funciona
- Menu mostrando os operadores disponíveis na calculadora, usando letras para cada um: A = SOMA, B = SUBTRAÇÃO, C = DIVISÃO e D = MULTIPLICAÇÃO
- Usuário escolhe qual opção de operador ele quer utilizar em sua conta, o primeiro e o segundo número que serão usados na operação (atribuídos em uma variável)
- O programa exibe o resultado e pergunta se ele quer calcular novamente

Como rodar
python Calculadora.py

Utilizei:
- While: para manter a calculadora rodando até o usuário decidir sair
- Condições (if/elif): para identificar qual operação o usuário escolheu e calcular o resultado correto
- Break: para encerrar o loop quando o usuário não deseja fazer outro cálculo
